# Workforce Attrition Analysis at Palo Alto Networks

## Project Overview

Employee attrition is one of the most significant challenges faced by organizations, impacting productivity, workforce stability, and operational costs. This project analyzes employee attrition at Palo Alto Networks using Exploratory Data Analysis (EDA) and Machine Learning to identify the major factors influencing employee turnover and provide data-driven HR recommendations.

---

## Problem Statement

The objective of this project is to analyze employee attrition patterns and identify the key factors contributing to workforce turnover. By leveraging HR analytics and predictive modeling, the project aims to support evidence-based decision-making for employee retention and workforce planning.

---

## Objectives

- Analyze employee attrition trends.
- Identify departments and job roles with higher attrition.
- Examine demographic, tenure, and workload-related factors.
- Calculate HR Key Performance Indicators (KPIs).
- Build a Machine Learning model to predict employee attrition.
- Generate actionable HR recommendations.

---
